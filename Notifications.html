Getting Started

Welcome to the ANG SILAKBO Admin Dashboard! This platform helps you manage and respond to user messages efficiently.

Here's how to get started:

  • The Messages section displays all incoming messages from users
  • The Users section shows message history grouped by user
  • Use filters and search to quickly find specific messages
  • Click on any message to view details and respond

Managing Messages

Efficiently handle messages with these steps:

  1. Click on a message to open its details in a modal
  2. Review the message content and user information
  3. Update the status to "In Progress" while working on it
  4. Add your response in the comment section
  5. Change status to "Resolved" when complete

Message statuses:

  • Received - New messages that need attention
  • In Progress - Messages you're currently working on
  • Resolved - Completed messages that require no further action

User History

The Users section helps you track communication history with each user:

  • View all messages from a specific user in one place
  • See patterns in user communication
  • Provide consistent responses based on previous interactions
  • Search for users by name or email

This feature is especially helpful for maintaining continuity when multiple admins are handling messages from the same user.

Search & Filter

Quickly find what you need with search and filters:

  • Use the search bar to find messages by subject, content, name, or tracking code
  • Filter messages by status (Received, In Progress, Resolved)
  • View only unread messages with the "Unread" filter
  • Reset to view all messages with the "All" filter

Mobile Usage

The dashboard is fully responsive for mobile devices:

  • Access the sidebar menu by tapping the menu icon in the top left
  • Scroll horizontally to view all filter options
  • Modals are optimized for smaller screens
  • All functions work the same as on desktop
